# Quizlet Clone

A flashcard and quiz learning platform similar to Quizlet.

## Tech Stack

Frontend:
- Next.js
- TailwindCSS
- shadcn/ui

Backend:
- Node.js
- Express.js

Database:
- sqlite
- Prisma ORM

## Core Features

- Create projects
- Create flashcards
- Flashcard study mode
- Quiz mode
- Save progress

## Folder Structure

/client => frontend
/server => backend
/docs => documentation
